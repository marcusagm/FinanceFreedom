# Índice de Recursos - Finance Freedom V1.0

Este documento lista todos os recursos detalhados para a Versão 1.0 do Finance Freedom.

## Módulos Principais

### [F01 - Dashboard & Hub Financeiro](Features/F01-Dashboard.md)

O painel de controle principal que oferece a visão imediata da saúde financeira ("Hoje").

### [F02 - Smart Import Manager (OFX & Email)](Features/F02-Connection-Manager.md)

**[REFACTORED]** O motor de ingestão de dados. Substitui conexões caras por importação inteligente de arquivos OFX e monitoramento de e-mail (IMAP) para custo zero.

### [F03 - Income Engine (Renda & Capacidade)](Features/F03-Income-Engine.md)

O motor de geração de riqueza. Permite modelar rendas fixas e variáveis.

### [F04 - Debt Strategy Engine (Alívio de Dívidas)](Features/F04-Debt-Engine.md)

O coração estratégico do sistema. Algoritmos de Snowball e Avalanche.

### [F05 - Transaction Manager (O Livro Razão)](Features/F05-Transaction-Manager.md)

Sistema de categorização e registro de transações com inteligência de aprendizado.

### [F06 - Simuladores Contextuais](Features/F06-Simulators.md)

Ferramentas de tomada de decisão embutidas no fluxo.
