# Visão do Produto Refinada: Finance Freedom

## 1. Visão Geral

**Finance Freedom** é o Ecossistema Financeiro Self-Hosted Definitivo. Ele substitui as mensalidades caras de apps bancários por uma inteligência local capaz de processar seus dados bancários (arquivos e e-mails) e traçar a rota para a independência financeira.

## 2. Proposta de Valor

"Pare de pagar para ver seu próprio dinheiro. Controle total, custo zero, privacidade absoluta."

## 3. As 3 Colunas do Produto (Core Pillars)

### A. Smart Import (O Fim da Digitação)

-   **Importação OFX:** Arraste e solte arquivos de qualquer banco.
-   **Email Sync:** O sistema monitora seu e-mail em busca de extratos e processa automaticamente.
-   **PDF Parser:** (Futuro) Leitura inteligente de extratos em PDF.

### B. Estratégia Ativa (Debt Slayer)

-   **Planejamento:** Snowball (Pagamento da menor dívida) e Avalanche (Pagamento do maior juro).
-   **Execução:** Acompanhamento visual do progresso.

### C. Income Engine (Capacity)

-   **Freelance Booster:** Use sua força de trabalho para acelerar a quitação.

## 4. Escopo da Versão 1.0 (MVP)

### Módulo 1: Data Ingestion (Novo Core)

-   [ ] **OFX Drag & Drop:** Interface simples para upload.
-   [ ] **IMAP Connector:** Conexão com conta de e-mail para baixar anexos automaticamente.
-   [ ] **Deduplicação Inteligente:** Evitar lançar a mesma transação duas vezes.

### Módulo 2: O Dashboard

-   [ ] Visão consolidada de múltiplas contas (agregadas via importação).
-   [ ] Alertas de "Dados Desatualizados" (ex: "Última importação há 5 dias").

### Módulo 3: O Estrategista

-   [ ] Gestão de Dívidas (Cadastro Manual).
-   [ ] Algoritmo Snowball.

## 5. Jornada do Usuário (Macro)

1.  **Setup:** Instala o Docker.
2.  **Ingestão:** Exporta OFX do banco (ou encaminha e-mail).
3.  **Mágica:** O sistema processa, categoriza e atualiza o saldo.
4.  **Ação:** O Dashboard mostra: "Seu saldo é X. Pague Y na dívida Z".
