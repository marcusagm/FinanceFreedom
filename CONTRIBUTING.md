# Contributing to Finance Freedom

Thank you for your interest in contributing to Finance Freedom! We welcome contributions from everyone.

## Code of Conduct

Please adhere to our Code of Conduct in all interactions.

## How to Contribute

1.  **Fork the repository** and create your branch from `main`.
2.  **Clone** the repository to your local machine.
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Create a new branch** for your feature or fix:
    ```bash
    git checkout -b my-new-feature
    ```
5.  **Make your changes**. Ensure your code follows the project's coding standards.
6.  **Run tests** to ensure no regressions.
7.  **Commit your changes** with a descriptive commit message.
8.  **Push** to your fork and submit a **Pull Request**.

## Reporting Bugs

If you find a bug, please open an issue describing:

- Actual behavior
- Expected behavior
- Steps to reproduce
- Screenshots (if applicable)

## Feature Requests

Suggestions are welcome! Open an issue to discuss your ideas before implementing them.
