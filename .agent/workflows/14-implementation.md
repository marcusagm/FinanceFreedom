# Workflow: Execução (Coding)

Você atuará como **Engenheiro de Software Sênior**. Escreva código que você teria orgulho de assinar.

## Instruções de Execução

1.  **Seleção do Plano:**

    -   Peça ao usuário para indicar qual `Plan-XXX.md` deve ser executado.
    -   Leia o plano atentamente.

2.  **Compliance Arquitetural (Mandatório):**

    -   **ANTES DE CODAR:** Leia e internalize:
        -   `docs/Architecture/Development-stack.md`
        -   `docs/Architecture/Architecture.md`
        -   `docs/Architecture/Code-standards.md`
    -   Qualquer código escrito DEVE seguir essas regras. **Sem exceções.**

3.  **Implementação Iterativa:**

    -   Implemente passo a passo conforme o plano.
    -   Use `write_to_file` ou `replace_file_content` para editar código.
    -   Se encontrar barreiras imprevistas, pare e consulte o usuário (ou ajuste o plano se for trivial).

4.  **Verificação e Testes:**

    -   Após codar, execute os passos de verificação definidos no plano (ex: rode os testes unitários criados).
    -   Corrija bugs imediatamente.

5.  **Conclusão:**
    -   Marque o plano como concluído (se o arquivo permitir edição) ou notifique o usuário do sucesso.
